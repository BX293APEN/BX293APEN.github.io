## 動詞

- 2回繰り返すと柔らかくなる

| 日本語 | 中国語 | ピンイン(発音) | ニュアンス・使い方 |
|---|---|---|---|
| 触る | 摸 | mō (モー) | 手で触れる、なでる。もっとも万能です。 |
| つねる | 捏 | niē (ニエ) | ほっぺを「もちもち」とつまむ時にもこれ。 |
| 撫でる | 抚摸 / 摸摸 | fǔmō / mōmō | 優しくなでるなら「抚摸」、よしよしなら「摸摸」。 |
| キスする | 亲 / 亲亲 | qīn (チン) | 「親(qīn)」の一文字でキスする(ちゅーする)です。 |
| ハグする | 抱 / 拥抱 | bào / yōngbào | おまけ。ギュッとする時はこれを使います。 |
| 〜と思う / 感じる | 觉得 | juéde (ジュエデ) | 自分の意見や感想を言う時の必須動詞。「〜な気がする」にも使えます。 |
| (エアコンを)つける | 开 | kāi (カイ) | エアコン(空调)や電気をつける、ドアを開ける時に使います。 |
| (エアコンを)消す | 关 | guān (グアン) | エアコンや電気を消す、ドアを閉める時に使います。 |
| (風に)当たる / 吹く | 吹 / 吹吹 | chuī (チュイ) | クーラーの風に当たって「気持ちいい〜」と言う時に使います。 |
| こする / 揉む | 揉 / 揉揉 | róu (ロウ) | 花粉で「目をこする(揉眼睛)」の動詞。優しく揉む時にも使えます。 |
| (薬を)飲む / 食べる | 吃 | chī (チー) | 中国語では薬(药)は「飲む」ではなく「食べる(吃)」と言うので注意！ |

## 副詞(動作のやり方・状態)

- 動詞の前に置く副詞

| 日本語 | 中国語 | ピンイン | ニュアンス・使い方 |
|---|---|---|---|
| 優しく / 穏やかに | 温柔地 | wēnróu de | 愛情や思いやりを持って、ソフトに振る舞う時。 |
| 優しく / そっと | 轻轻地 | qīngqīng de | 力を入れず、触れるか触れないかのように(物理的に軽く)。 |
| ゆっくりと | 慢慢地 | mànmàn de | 時間をかけて、焦らずに動作を行う時。 |
| ぎゅっと / しっかり | 紧紧地 | jǐnjǐn de | 力を込めて、密着させるように(ハグや手を握る時など)。 |
| 嬉しそうに | 高兴地 | gāoxìng de | 喜びを表情や態度に出しながら動作を行う時。 |
| 静かに / そっと | 安静地 | ānjìng de | 音を立てず、落ち着いた状態で行う時。 |

- 程度副詞

| 日本語 | 中国語 | ピンイン | 使い方・ニュアンス |
|---|---|---|---|
| (特に意味のない)とても | 很 | hěn | ※何もない時は絶対付けること <br> 形容詞を使うときの基本セット。基本は訳さなくてOK。 |
| とても / 非常に | 非常 | fēicháng | 「とても」をはっきり強調したいとき。 |
| すごく / めっちゃ | 太 A 了 | tài …… le | 「太痛了(痛すぎる/めっちゃ痛い)」のように、「了」と挟んで使います。 |
| ちょっと / 少し | 有点儿 / 有点 | yǒudiǎnr / yǒudiǎn | 「有点痛(ちょっと痛い)」のように、マイナスな状態に伝えます。 |

## 文構造

- 絶対主語を省略しないこと(翻訳機にかけると命令形になる)

### SVO
我亲妈妈

#### SV [(M)O]
我亲妈妈的脸

#### S [(M)V] O
我温柔地亲妈妈

### V了 (過去形)
我亲了妈妈

### S被OV了 (受動態)
我被妈妈亲了

### 想V (願望)
我想亲妈妈

### SVOC
我觉得妈妈很可爱
