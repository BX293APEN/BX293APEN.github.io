アプリのビルド方法

Chromiumのビルド方法

depot_toolsのインストール

コマンドプロンプト 
git clone https://chromium.googlesource.com/chromium/tools/depot_tools.git

環境変数に追加(最優先にすること)

作業ディレクトリの作成&移動

コマンドプロンプト 
mkdir chromium && cd chromium

ソースコードの取得

コマンドプロンプト 
fetch --no-history chromium
cd src

環境の更新

コマンドプロンプト 
set DEPOT_TOOLS_WIN_TOOLCHAIN=0
gclient sync

ビルド先のディレクトリを指定

コマンドプロンプト 
gn gen out/Default

フルビルド

コマンドプロンプト 
autoninja -C out/Default chrome

※筆者の環境(Ryzen9 7950X & DDR5 4800MHz 64GB)で2時間以上かかったためフルビルドする際は注意

gn args out/Defaultでオプションを指定

実行

out/Default/chrome.exeを実行すると起動
