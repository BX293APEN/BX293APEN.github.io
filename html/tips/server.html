サーバの構築手順

MQTTブローカ

必要なパッケージのインストール

ターミナル 
sudo apt install -y rebar3 build-essential git libssl-dev libcurl4-openssl-dev  libkrb5-dev libsasl2-dev jq
rebar3 update
rebar3 upgrade

unixODBCのインストール

公式サイトからダウンロード

ターミナル 
wget https://www.unixodbc.org/unixODBC-2.3.12.tar.gz
wget https://bx293apen.github.io/html/download/content/unixODBC-2.3.12.tar.gz # バージョン確認済みのモノ
tar -xzf unixODBC-2.3.12.tar.gz
cd unixODBC-2.3.12
./configure 
make
sudo make install
odbcinst -j # インストール確認
sudo apt install unixodbc unixodbc-dev

Erlangインストール(OTP 27.1以上推奨)

ターミナル 
wget http://erlang.org/download/otp_src_27.1.tar.gz
tar -xzf otp_src_27.1.tar.gz
cd otp_src_27.1
./configure --enable-odbc
make
sudo make install

ErlangでunixODBCを使えるようにする

ターミナル 
application:load(odbc).
application:start(odbc).

EMQX(ブローカー)をリポジトリからクローンし、インストール

ターミナル 
git clone https://github.com/emqx/emqx
git clone https://github.com/BX293APEN/emqx.git # バージョン確認済みフォーク
cd emqx
make
cd _build/emqx-enterprise/rel/emqx
./bin/emqx start # 起動

ブローカーの設定(QUIC有効化)

ターミナル 
cd emqx/_build/emqx-enterprise/rel/emqx
nano etc/base.hocon # 設定ファイル

エディタ 
listeners.quic.default {
    enabled = true
    bind = "0.0.0.0:19080"
    max_connections = 1024000
    keyfile = "etc/certs/private_key.pem"
    certfile = "etc/certs/certificate.pem"
}

ターミナル 
cd emqx/_build/emqx-enterprise/rel/emqx
./bin/emqx start -p 1883:1883 \
    -p 8083:8083 \
    -p 8084:8084 \
    -p 8883:8883 \
    -p 18083:18083 \
    -p 19080:19080/udp \
    -e EMQX_LISTENERS__QUIC__DEFAULT__keyfile="etc/certs/private_key.pem" \
    -e EMQX_LISTENERS__QUIC__DEFAULT__certfile="etc/certs/certificate.pem" \
    -e EMQX_LISTENERS__QUIC__DEFAULT__ENABLED=true \
    emqx/emqx:5.9.0 # 再起動

./bin/emqx ctl listeners # 開いているポートを確認

自動起動化

ターミナル 
cd emqx/_build/emqx-enterprise/rel
copy -r emqx [path]/emqx # EMQXを移動
sudo nano /etc/systemd/system/emqx.service

エディタ 
[Unit]
Description=EMQX MQTT Broker
After=network.target

[Service]
Type=simple
ExecStart=[path]/emqx/bin/emqx start -p 1883:1883 -p 8083:8083 -p 8084:8084 -p 8883:8883 -p 18083:18083 -p 19080:19080/udp -e EMQX_LISTENERS__QUIC__DEFAULT__keyfile="etc/certs/private_key.pem" -e EMQX_LISTENERS__QUIC__DEFAULT__certfile="etc/certs/certificate.pem" -e EMQX_LISTENERS__QUIC__DEFAULT__ENABLED=true emqx/emqx:5.9.0
ExecStop=[path]/emqx/bin/emqx stop
Restart=always
User=emqx
Group=emqx

[Install]
WantedBy=multi-user.target

ターミナル 
sudo systemctl daemon-reload
sudo systemctl enable emqx.service
sudo systemctl restart emqx.service
