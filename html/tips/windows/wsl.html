How to use WSL?

WSLをインストールする

コマンドプロンプト 
wsl --install
wsl --update

デフォルトをWSL2に設定する

コマンドプロンプト
wsl --set-default-version 2

WSLを再起動する

コマンドプロンプト
wsl --shutdown

デフォルトのディストリビューションを変える

コマンドプロンプト 
wsl -l -v      # ディストリビューションの一覧を表示
wsl --set-default [デフォルトにしたいディストリビューション名]

パスワードを忘れた

コマンドプロンプト 
wsl -u root    # 管理者権限でWSLを開く
passwd [パスワードを忘れたユーザ名]

WSLの設定

コマンドプロンプト 
notepad %userprofile%\.wslconfig

エディタ 
[wsl2]
memory=4GB                      # WSL2が使う最大メモリ
processors=4                    # 使用するCPUコア数
swap=8GB                        # スワップファイルのサイズ
localhostForwarding=true        # Windows⇔Linux間のローカル通信を許可 (デフォルト : true)
nestedVirtualization=true       # Docker等の入れ子仮想化を許可
guiApplications=true            # GUIアプリを有効化 (デフォルト : true)
networkingMode=default          # ミラーリングモード設定 
                                # default : 分離仮想ネットワーク mirrored : ミラーリングモード

[experimental]
# autoMemoryReclaim=gradual     # 未使用メモリの自動解放
# sparseVhd=true                # WSLディスクの自動縮小を有効化
