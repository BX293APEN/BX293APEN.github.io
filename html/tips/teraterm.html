Tera Termマクロ

Tera Term

 Tera Termは、Windows用のターミナルエミュレータである。
 ここでは、Tera Termが保存されているパスを C:\Program Files (x86)\teraterm\ とする。

SSHマクロでログインする

LOGIN.ttl 
HOSTADDR = "[ターゲットのIPアドレス]"
USERNAME = "[ユーザ名]"
PASSWORD = "[パスワード]"

sprintf2 COMMAND "%s:22 /ssh /2 /auth=password /user=%s /passwd=%s" HOSTADDR USERNAME PASSWORD
connect COMMAND

end

バッチファイル (拡張子 : .bat) 
@echo off
cls
"C:\Program Files (x86)\teraterm\ttpmacro.exe" "%~dp0\LOGIN.ttl"
