SSH

公開鍵認証

Windows ターミナル 
ssh-keygen -t ed25519 -C '""'
cd %userprofile%/.ssh
scp id_ed25519.pub [リモート接続するユーザ名]@[リモート接続する端末のIPアドレス]:~/

ターミナル 
cd ~/
mkdir .ssh
mv id_ed25519.pub .ssh/authorized_keys
chmod 700 .ssh
cd .ssh
chmod 600 authorized_keys
